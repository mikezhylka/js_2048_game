- **Goal**: Slide numbered tiles on a 4x4 grid to combine them into a tile with the value of **2048**.
- **How to Play**: Use arrow keys to move tiles. Tiles with the same value merge, creating a new tile with double the value. Keep combining tiles until you reach **2048**â€”or run out of moves!
- **Win/Loss**: Win by reaching 2048. The game is over when no valid moves remain.
